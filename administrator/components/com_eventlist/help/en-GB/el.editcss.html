CSS-Editor

Description

Edit in this screen the CSS file of EventList.



Attention
Don't change anything if you don't know exactly what you do. Read the comments carefully and backup this file before you alter it.

Navigation

To access this screen use: Components -> EventList -> Edit CSS

User Access Level

Super Administrator

Toolbar Icons

Apply: Saves the css file and returns to this screen.

Save: Saves the css file and returns to the EventList screen.

Cancel: Go Back to the EventList screen.

Help: Shows this Help screen.

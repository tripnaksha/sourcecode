List Events

Description

Manage your event items from that screen. Here you may view a listing of all events. You can delete, edit or add events using this screen. Also you can publish or unpublish events and manage users who has registered to it.


Navigation

To access this screen use: Components -> EventList -> Events

User Access Level

Super Administrator Administrator Manager

Toolbar Icons

Select Event(s) by clicking in the checkbox(es) at the beginning of each row.

Archive: Click the 'Archive' icon to put the Event items into an archived state. They are still accessible on the EventList Frontpage if you select in the Settings 'Archive old Events'.

Publish and Unpublish: Click the 'Publish' or 'Unpublish' icons to publish or unpublish the selected event items. If you have unpublished an event it is not longer visible on the Frontpage.

New: Click the 'New' icon to create a new event.

Edit: Click the 'Edit' icon to change the selected event item. You can also do this by clicking the name.

Remove: Click the 'Remove' icon to delete the selected event Items.

Copy: Click to copy an event.

Help: Click the Help icon to access a relevant Help Screen.

Columns:

#: The number of the entry in the list.

Checkbox: Select the item or items to be edited or deleted by clicking in this box.

Date: This is the date of the Event.

Sometimes a padlock icon next to it shows that it is checked-out. To check-in, click the date to edit. You will not be able to edit an Event item if it is checked out by another user.
It must be checked back by an administrator. The administrator navigates to the menu item Tools -> Global Checkin in the Menu bar. It can also be checked back in by the User who has checked the item out by clicking the 'Edit' icon of the Event item and then clicking either the 'Save' or 'Cancel' icon.

Title: This is the title of the Event item.

Event Time: This is the Time when the Event starts.

Venue: This is the venue location where the Event will be held.

Category: This is the category the Event belongs to.

City: This is the city where the Event will be held.

Published: This shows whether the Event item is published() or unpublished(). Click the icon to change the state.

User Event: This column shows the details of the user who has submitted an event from the Frontpage.

Reg. Users: This shows the number of Users who has registered for an event. Click the number if you are interested in more detailed information about them or you want to remove some of them.

Other Functions

Search: You may enter text in the "Filter" box to refine the list based on whether the text is found in the title of any event item. Type in any text and press Enter or click the go button. You also can filter the search by limit it to the venue, category, city or event title.

Drop Down: Select the number of list entries displayed per page [Options between 5 – 100].

Navigation Buttons: Use Start, Previous, Next, and End buttons to navigate between list pages.

Page Indicator: Shows which page is displayed, in the form of page n of n.

Ordering: Click on the tableheader to order the items by name, state or date (depends on the format of the column).

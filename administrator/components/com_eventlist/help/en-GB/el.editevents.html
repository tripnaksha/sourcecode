Edit Event

Description

Edit your event items or add new ones from that screen.



Tip
Use Plugins to add additional functionality to EventList. You can connect your events for example with a reservationsystem like "akreservations' or Galleries with the help of Plugins.

Navigation

To access this screen use: Components -> EventList -> Add Event or by clicking an event in the overview

User Access Level

Super Administrator Administrator Manager

Toolbar Icons

Apply: Saves the event and returns to the current view.

Save: Saves the event and returns to the overview.

Cancel: Cancels the action. Any modification is lost. If an item is opened to edit, the event Item remains checked-in.

Help: Opens this help screen.


Main publishing area

Title: Enter or edit (if needed) the title of the Event here.

Alias: Enter or edit (if needed) the alias of the Event here. The alias is used in the url, only use url safe characters here. Leave this field blank if you want EventList to create it automatically.

Published: Select Yes to publish the event or No to unpublish. Published events are displayed on the Frontend.

Venue: Choose the venue you want to associate with this event or set the value to no venue. If you want you can also add a new venue to the venue database by clicking the "add" button.

Category: Choose the category you want the item to belong to. The dropdown list will display the available Categories.

Textfield: This is the Event description. You can use Joomla Plugins in this field.

Advanced publishing area

Eventdate: Select the Eventdate. (required)

Enddate: Select the Date when the Event ends.

Time: The time when the Event starts.

Endtime: The time when the Event ends.

Enable registration: Select yes if users should be able to register to this Event.

Allow unregistration: Select yes if users should be able to unregister/cancel their registration from this Event.

Image: Choose an image for this Event (flyer) or upload it.

Reoccuring Events: Choose the rythm of the reoccurance: dayly, weekly, monthly, yearly or at a single weekday of the month. After that choose the timeframe of the reoccurance.

Metadata information: Enter or edit (if needed) the Meta Description and the Meta Keywords of the event here. You may use the suggestions in the settings to set default values which will be automatically set to the event data.

Update check

Tip
Check always for updates. Updating keeps your site secure and mostly provides new useful functionality to you

If you don't visit the home of EventList, Schlu.net, from time to time or you don't use the update notification service you should click the button "Updatecheck" often to see if a new version of EventList is available.

During the update check a file located at schlu.net is analyzed. The data contained in that file is shown after in a popup including a notice if you are using the latest version of EventList or not.
Keeping EventList up to date is very important. Beside the strongest point, security fixes if needed, updates also often provides new features and fixes for bugs.

CSS-Editor

Beschreibung

In dieser Ansicht können Sie die von EventList verwendete CSS Datei bearbeiten.



Achtung
Verändern Sie nichts, wenn Sie nicht genau wissen was Sie tun. Lesen Sie die Kommentare sorgfältig und erstellen Sie ein Backup der Datei bevor Sie diese verändern.

Navigation

Folgendermaßen können Sie auf diese Ansicht zugreifen: Komponenten -> EventList -> CSS bearbeiten

Benutzer Zugriffslevel

Super Administrator

Toolbar Icons

Hinzufügen: Speichert die CSS-Datei, ohne diese Ansicht zu verlassen.

Speichern: Speichert die CSS-Date und wechselt zur EventList-Startseite.

Abbrechen: Wechselt zur EventList-Startseite ohne zu speichern.

Hilfe: Klicken Sie das "Hilfe"-Icon, um eine relevante Hilfe zu öffnen.
